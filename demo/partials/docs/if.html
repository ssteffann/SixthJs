data-bind:if

Binding type: if deletes or recreates a portion of the DOM tree base on given property. If property asigned to binding type if is evaluated to false value, then all children elements are removed from from DOM tree, in other case a copy of the elements is inserted in DOM and all bindings are restored.

Binding type: if differs from CSS class "show" or "hide", because it doesn;t use CSS properties like display: none. It completely removes and recreates the elements in DOM.

Usasge:

data-bind: if is used as attribute:

          
            <div data-bind="{if: property }">
              <textarea></textarea>
              <p>Hello World!</p>
            </div>

Example:

          
            <div data-controller="ExampleController">
              <input type="checkbox" data-bind="{model: property}"/>

              <div data-bind="{if: property }">
                <textarea data-bind="{model: action}"></textarea>
                <p>Hello World!</p>
                <p>{{ action }}</p>
              </div>
            </div>

            <script>
              sixth.controller('ExampleController', function(){
                this.property = true;
              });
            </script>
