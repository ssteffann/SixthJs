data-bind:include

Binding type: include make a HTTP get request through Ajax for an external HTML partial, and then include and compiles it. The HTML partial is cached to reduce the number of request. By default all templates are restricted to the same domain. Templates are bound to the parent controller.

This binding type get as argument url of HTML fragment. The url can be a complete one or relative, if you use relative url it must be relative to root location.

Usage:

data-bind: include is used as attribute:

          <div data-bind="{include: partials/mytemplate.html }"></div>
