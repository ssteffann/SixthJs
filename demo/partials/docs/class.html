data-bind:class

Binding type: class help developer to dynamically set CSS class on an HTML element. Attribute receive an object where is specified the class and the property from scope. Class is added once evaluated the property is true.

Binding type: class won't add duplicate classes if the class is allready set. This type also don't have any limitation related to element type, and can be used with any of it.

Usage:

data-bind: class is used as attribute:

      
        <div data-bind="{class: { className: property }}">
          <p class="red" data-bind="{class: { className: property }}">
            Some Text!
          </p">
        </div>

Example:

          
          <div data-controller="ExampleController" >
            <input type="checkbox" data-bind="{model: property}"/>

            <p data-bind={class={ text-danger: property }}></p>
          </div>

          <script>
            sixth.controller('ExampleController', function(){
              this.property = false;
            });
          </script>
