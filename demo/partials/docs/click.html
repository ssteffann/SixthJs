data-bind:click/dblclick

Binding type: clickand dblclick allows developer to specify a custom action on click or dbclick event. This two types can be attached on input type elements, buttons, selects, textarea and anchors.

Usage:

data-bind: click/dblclick is used as attribute:

          
            <button data-bind="{click: funcName }">Click me </button>
            <a href="/address" data-bind="{click: funcName }">Click link </a>

            <button data-bind="{dbclick: funcName }">Click me </button>

Example:

          
          <div data-controller="ExampleController" >
            <button data-bind="{click: myFunction }">Click me </button>
            <button data-bind="{dblclick: myFunction }">Double click me</button>
          </div>

          <script>
            sixth.controller('ExampleController', function(){
              this.counter = 1;

              this.myFunction = function(){
                this.counter += 1;
              };
            });
          </script>
