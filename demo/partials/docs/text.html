data-bind:text {{}}

Binding type: text is not exposed, and is not used with attribute data-bind. As an alias for this binding type is used markup {{ }}. All strings with this markup will be compiled into an interpolation function. This type provides one way binding and cand be used in text contend and also in elements attributes.

Usasge:

data-bind:text as markup in text:

          
            <p>Hello my name is {{ client.name }}!!!</p>

data-bind:text as markup in attribute:

          
            <a href="clients/{{ client.name }}">Find me here!</a>

Example:

          
            <div data-controller="ExampleController" >
              <p>Hello my name is {{ client.name }}!!!</p>
            </div>

            <script>
              sixth.controller('ExampleController', function(){
                this.client = {
                  name: 'Borg'
                };
              });
            </script>
