data-bind:change

Binding type: change trigger given function when user changes the input. The event is triggered at the end of the change. Will throw an error if given expression is not a function. Also will not trigger if attached model is changed programmatically and not from input.

Usage:

data-bind: change is used as attribute:

          
              <input type="text" data-bind="{change: myFunction}"/>

              <select data-bind="{change: myFunction}"></select>

              <textarea data-bind="{change: myFunction}"></textarea>

Example:

          
              <div data-controller="ExampleController" >
               <input type="text" data-bind="{change: myFunction}"/>
              </div>

              <script>
                sixth.controller('ExampleController', function(){
                 this.counter = 1;

                 this.myFunction = function(){
                  this.counter += 1;
                 };
                });
             </script>
