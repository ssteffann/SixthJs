data-bind:model

Binding type: model works with elements input,textarea, select and have the role of binding and expose a property from data-controller scope.

data-binding: model respond for:
  • Binding elements like input,select, textarea from view into the model.

data-binding: model will try to bind given property to the scope by evaluating it. If the property is not defined on current scope, it will be ignored.

By default data-bind: model spy model by reference, not value, this is important to know when binding inputs to model that are objects. If only properties of the object change, watcher will not trigger and the input will not be re-rendered.

Usage:

data-bind: model is used as attribute:

          
            <input type="text" data-bind="{model: property}"/>

            <select data-bind="{model: property}"></select>

            <textarea data-bind="{model: property}"></textarea>

Example:

          
            <div data-controller="ExampleController" >
              <form name="myForm">
                Name:
                <input type="text"
                          data-bind="{model: client.name}"/>
              </form>
              Result: user.name = {{ client.name }}
            </div>

            <script>
              sixth.controller('ExampleController', function(){
                this.client = {
                  name: 'SixthJs'
                };
              });
            </script>
Result:

user.name = {{ user.name }}
